# Shopify App Store Listing Draft

## App Name

Withdrawal Portal

## Short Description

Online withdrawal request flow for EU stores.

## Tagline

Help customers submit withdrawal declarations from your storefront.

## Long Description

Withdrawal Portal helps merchants provide a visible online withdrawal request function on their Shopify storefront.

Customers enter their order number and email address, the app verifies the order through Shopify, and eligible line items can be selected for withdrawal. The app stores the submitted request with selected items, timestamps, and audit events, then sends customer confirmation and merchant notification emails.

The app is designed for EU-first storefront workflows with Slovak, Czech, and English customer-facing copy. It includes a theme app extension for storefront placement and a merchant dashboard for reviewing requests and exporting records.

Withdrawal Portal helps with the technical workflow. It is not legal advice and does not guarantee legal compliance.

## Feature Bullets

- Storefront withdrawal form through Shopify app proxy.
- Order lookup by order number and exact customer email match.
- Configurable self-service cutoff window.
- Customer confirmation and merchant notification emails.
- Request dashboard, detail pages, audit events, and CSV export.
- Theme app extension block with SK/CZ/EN labels.
